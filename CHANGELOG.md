# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-15

### Added

- Initial release
- Pre-push Git hook functionality
- Auto-installation on npm install
- Configurable commands and options
- Support for multiple commands
- CI detection and skip functionality
- Branch-specific skip functionality
- Timeout support for long-running commands
- Verbose mode for debugging
- TypeScript support
- Color-coded terminal output
- Comprehensive CLI commands (install, uninstall, run, help)
- Configuration via package.json or .nopushoopsrc.json
